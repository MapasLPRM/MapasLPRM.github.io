O Projeto

Conhecendo a comunidade para saber como melhorá-la

O projeto Anotações colaborativas de mapas Web, visa criar uma base de dados multimídia com imagens,vídeos e comentários georreferenciados e relacioná-los com um mapa Web sobre o qual elas possam ser visualizadas. A Coleta de dados pode ser realizada de várias formas, desde comentários diretos sob o mapa Web, até o uso de informações multimídias(com suas respectivas coordenadas geográficas) que são coletadas por um smartphone e integradas automaticamente ao mapa usando a plataforma Searchlight. As coletas terão três alvos previamente definidos, que são:

Arte ou sujeira: Os alunos procurarão por expressões gráficas em paredes e muros da cidade em que vivem e serão classificados pelos próprios alunos envolvidos proporcionando um mapa Web de arte e outro de sujeira.

Pontos de acúmulo de lixo: Os alunos irão procurar por pontos de acúmulo de lixo, tirarão fotos e anotarão informações importantes, como o tipo de lixo, local onde foi tirada a foto, de modo que possam voltar no local posteriormente e verificar se houve acúmulo de lixo, o que seria usado para determinar pontos de reincidência de lixo. Tais informações podem ser fornecidas a prefeitura da cidade para que as mesma possa usar os dados para guiar campanhas de conscientização e fiscalização.

Focos de mosquito: Os alunos irão identificar possíveis focos de mosquito para posteriormente se agruparem e juntos determinarem se os pontos identificados podem realmente ser focos para reprodução de mosquitos. Os dados estarão disponíveis para toda a comunidade interessada, inclusive órgãos de competência para que precauções sejam tomadas.
